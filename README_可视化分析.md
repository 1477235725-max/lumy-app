# 星世界地图商业化数据分析 - 使用说明

## 📊 概述

本项目包含多个用于分析"游玩排名前300地图 付费&广告.xlsx"数据的脚本，可以进行全面的商业化分析和数据可视化。

## 📁 文件说明

### 主要分析脚本

1. **starworld_visualization.py** ⭐ 推荐
   - 功能：生成详细的数据可视化图表
   - 输出：PNG 格式图表（300 DPI 高清）
   - 包含：数据概览、付费分析、广告分析、排名分析、商业化综合仪表板

2. **detailed_commercial_analysis.py**
   - 功能：生成详细的商业化分析文本报告
   - 输出：TXT 格式分析报告
   - 包含：付费分析、广告分析、用户行为、商业指标、商业化建议

3. **analyze_now.py** / **run_analysis.py**
   - 功能：基础数据分析
   - 输出：TXT 格式摘要报告

4. **simple_read.py**
   - 功能：简单数据读取和概览
   - 输出：TXT 格式数据摘要

## 📊 商业化分析指标

### 付费相关指标
- **总收入**：所有付费金额的总和
- **ARPU**：每用户平均收入（Average Revenue Per User）
- **付费转化率**：付费用户占总用户的比例
- **付费用户数**：有付费行为的用户数量
- **平均付费金额**：付费用户的平均付费金额

### 广告相关指标
- **曝光量**：广告展示的总次数
- **点击量**：广告被点击的总次数
- **CTR（点击率）**：点击量/曝光量
- **CPM（千次展示成本）**：广告每千次展示的成本
- **CPC（单次点击成本）**：广告每次点击的成本

### 用户行为指标
- **活跃用户数**：活跃用户的数量
- **用户留存率**：用户留存的比例
- **用户时长**：用户平均使用时长

### 排名相关指标
- **平均排名**：地图的平均排名
- **Top 10 占比**：排名前10的地图数量
- **排名分布**：各排名区间的地图数量

## 🎯 商业化效果评估

### 评分维度
1. **付费能力**（0-100分）
   - 基于总收入计算
   - 基准：10万元为满分

2. **广告效果**（0-100分）
   - 基于广告指标计算
   - 考虑曝光、点击、CTR等

3. **用户活跃**（0-100分）
   - 基于活跃用户数计算
   - 基准：1万活跃用户为满分

4. **排名表现**（0-100分）
   - 基于平均排名计算
   - 排名越小，得分越高

### 综合评分
- 各维度评分的算术平均值
- 用于评估整体商业化效果
